# Kvitra

Kvitra er en webapp, der hjælper brugere med at få overblik over deres forbrug.

Målet er at gøre det nemt at uploade kvitteringer, lade AI analysere dem og vise tydelige indsigter om, hvad man bruger sine penge på.

Projektet fokuserer på:
- login og brugerprofiler
- upload af kvitteringer
- AI-baseret scanning og kategorisering
- dashboard med forbrugsindsigter

Kvitra er bygget som et moderne React/Next.js projekt med Supabase til authentication og backend.
